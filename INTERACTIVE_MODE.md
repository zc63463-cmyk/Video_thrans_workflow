# 交互式菜单使用指南

Video Collector 现在支持交互式菜单，无需记住复杂的命令行参数！

## 🚀 启动方式

### 方式 1：无参数运行（推荐）

```bash
python scripts/main.py
```

会自动显示交互式菜单：

```
+---------------------+
|     Video Collector |
|     视频收藏 -> AI 输入包生成工具 |
|     支持 Bilibili / YouTube / 本地文件 |
+---------------------+

请选择操作：

+---------------------------------------+
|   1      | 导出 AI 输入包（单视频）   |
|----------+----------------------------|
|   2      | 导出 AI 输入包（批量 URL） |
|----------+----------------------------|
|   3      | 处理本地音视频文件         |
|----------+----------------------------|
|   4      | 同步收藏夹                 |
|----------+----------------------------|
|   5      | 重新生成笔记               |
|----------+----------------------------|
|   h      | 显示帮助                   |
|----------+----------------------------|
|   q      | 退出                       |
+---------------------------------------+

请输入选项 [1/2/3/4/5/h/q] (1):
```

### 方式 2：传统命令行参数

```bash
# 单视频导出
python scripts/main.py --bundle "https://www.bilibili.com/video/BVxxxxxxxxxx/"

# 批量处理
python scripts/main.py --bundle --url-file urls.txt --workers 5

# 本地文件
python scripts/main.py --media-file "C:\path\to\episode.mp3"
```

## 📋 菜单选项说明

### 1. 导出 AI 输入包（单视频）

**功能**：处理单个视频（Bilibili/YouTube），生成 AI 可处理的输入包。

**输入引导**：
- 视频链接（可选，直接回车从剪贴板读取）
- 是否强制使用 Whisper 转录？
- Whisper 模型大小（如果启用转录）
- 语言（如果启用转录）
- 是否启用缓存？

**输出**：
```
output/bundles/<platform>-<video_id>-<title>/
  metadata.json          # 视频元数据
  raw.json              # 原始抓取数据
  source.md             # 原始资料 Markdown
  transcript.txt        # 转录文本
  transcript_segments.json  # 分段转录
  transcript_timestamps.md  # 带时间戳的转录
  prompt.md             # 给大模型的输入包 ⭐
```

### 2. 导出 AI 输入包（批量 URL）

**功能**：从文件读取多个 URL，并发处理。

**输入引导**：
- URL 文件路径（每行一个 URL）
- 并发线程数（默认 3）
- 是否强制使用 Whisper 转录？
- 是否启用缓存？

**示例 `urls.txt`**：
```
https://www.bilibili.com/video/BV1xx411c7mD/
https://www.bilibili.com/video/BV1xx411c7mE/
https://www.youtube.com/watch?v=xxxxxxx
```

### 3. 处理本地音视频文件

**功能**：处理本地音视频文件，用 Whisper 转录。

**输入引导**：
- 音视频文件路径
- Whisper 模型大小
- 语言

**适用场景**：
- 播客录音
- 会议记录
- 讲座视频

### 4. 同步收藏夹

**功能**：同步 Bilibili/YouTube 收藏夹到本地数据库，并生成 Obsidian 笔记。

**输入引导**：
- 平台（bilibili/youtube/all）
- 是否启用缓存？

**注意**：需要配置 cookies 才能访问收藏夹。

### 5. 重新生成笔记

**功能**：从数据库重新生成所有笔记（不重新抓取）。

**适用场景**：
- 修改了笔记模板
- 想要重新生成所有笔记

### h. 显示帮助

显示详细的使用帮助信息。

### q. 退出

退出程序。

## 🤔 常见问题

### Q1：如何退出交互式模式？

输入 `q` 或直接按 `Ctrl+C`。

### Q2：交互式模式和命令行参数可以混用吗？

可以！命令行参数优先级更高。例如：
```bash
# 使用命令行参数指定 URL，但其他参数走交互式
python scripts/main.py --bundle "https://..."
```

### Q3：交互式模式支持所有功能吗？

是的！交互式模式支持所有功能，并且会引导你输入必要的参数。

### Q4：如果 rich 不可用怎么办？

程序会自动降级到标准输入输出，但交互式菜单需要 rich 支持。如果 rich 不可用，请使用命令行参数。

## 📁 相关文件

- `scripts/main.py` - 主程序（包含交互式菜单逻辑）
- `config/credentials.yaml` - 配置文件（可选）
- `output/bundles/` - Bundle 输出目录
- `output/notes/` - 笔记输出目录

## 🚀 快速开始

1. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```

2. **运行程序**：
   ```bash
   python scripts/main.py
   ```

3. **选择操作**：
   - 输入 `1` - 处理单个视频
   - 输入 `2` - 批量处理视频
   - 输入 `3` - 处理本地文件
   - 输入 `h` - 查看帮助
   - 输入 `q` - 退出

4. **按照提示输入参数**

就这么简单！🎉
